
# Nuxt starter :fire:

  

> Nuxt starter is a custom starter that allows you to make website and applications. It use the Vue.js framework called Nuxt.js. It is very useful to create a website from an API, like the WordPress REST API.

## Features

- Automatic transpilation and bundling (with webpack and babel)
- Hot code reloading
- Server-side rendering (SSR)
- Serving data with WordPress REST API
- UI Components (Button, card etc...)
- Page transition
- Global CSS
