# COMPONENTS

The components directory contains your Vue.js Components.

All Components are auto imported in pages files

## Conventions

- All files are named using PascalCase convention
- If component is unique, prefix him with "The" ex : "TheHeader.vue"
- Scope css in component file

Folder structure: 

- /layout : General files used in many files but writed only one time (file in this folder have "The" prefix)
- /ui : UI components used to build our application
- /utils : Utils files to avoid redundancy
